Name

Html5.upload

Type

Plugin

Describe

This is a upload plugin based on html5 fileAPI(FileReader), it support any MIME types of file upload. Design your own script to complete upload task by html5Upload plugin.

Feature

  1. Multiple MIME types supported.
  2. Read uploaded file through FileReader API.
  3. Custom script to complete uploaded files.

Base on

HTML5, Javascript, JQuery
